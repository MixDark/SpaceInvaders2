# Space invaders 2.0

Una versión moderna y frenética del clásico arcade, desarrollada en Python con Pygame. Enfréntate a 31 niveles de invasión alienígena, luchando contra jefes colosales y desbloqueando un arsenal devastador.

## Características

- **31 niveles de acción intensa**: Dificultad progresiva con oleadas dinámicas.
- **Batallas contra jefes**: Encuentros épicos en los niveles 11, 21 y 31 con mecánicas únicas.
- **Sistema de armas avanzado**:
  - Disparos dobles, triples y quíntuples.
  - Láser perforante y misiles teledirigidos.
  - Disparo 'ripple' expansivo y mega rayo devastador.
- **Power-ups acumulables**: Recoge mejoras para potenciar tu nave temporalmente.
- **Banda sonora épica**: Música dinámica de la saga Gradius que cambia con cada nivel.
- **Modo piedad**: Sistema de ayuda en batallas de jefes para garantizar la diversión.

## Controles

- **Flechas direccionales**: Mover la nave.
- **Espacio**: Disparar (mantener para fuego automático).
- **Teclas 1-9**: Activar armas especiales desbloqueadas (trucos).
- **P**: Pausar el juego.
- **R**: Reiniciar tras game over.

## Instalación

1. Asegúrate de tener Python instalado.
2. Instala las dependencias:
   ```bash
   pip install pygame
   ```
3. Ejecuta el juego:
   ```bash
   python main.py
   ```
